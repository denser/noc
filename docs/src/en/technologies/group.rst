=====
Group
=====

.. contents:: On this page
    :local:
    :backlinks: none
    :depth: 1
    :class: singlecol

Arbitrary grouping element for :ref:`Resource Groups<reference-resource-group>`.
Has no any specific meaning. Used to organize other groups in hierarchy.

Properties
----------

+----------------+-------+
| Property       | Value |
+================+=======+
| Service        | None  |
| Client         | None  |
| Single Service | False |
| Single Client  | False |
| Allow Children | True  |
+----------------+-------+
